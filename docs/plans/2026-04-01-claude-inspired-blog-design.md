# Claude-Inspired Personal Blog Design

## Goal
将博客改造成参考 Claude Blog 的编辑型界面，但保留个人博客语境，让它看起来像一位长期写作的工程师/研究者的出版空间，而不是企业产品官网。

## Design Direction
- 参考 Claude Blog 的暖白底、低饱和中性色、圆角卡片、强留白、克制边框和“编辑部”式排版。
- 不追求 1:1 复制，而是提炼其视觉语言并适配个人博客。
- 首页、文章列表页、详情页统一到同一设计系统中。

## Experience Principles
- 首页直接承担“浏览与发现文章”的任务，而不是只放一段简单介绍。
- 搜索、筛选、Grid/List 切换应成为核心交互。
- 文章详情页要有更强的阅读感和出版物气质，减少默认主题味道。

## Layout
- 顶部：简洁导航和品牌标识。
- Hero：大标题、简洁介绍、可选的个人定位说明。
- Browse 区：
  - 左侧筛选栏，包含分类、标签、年份等。
  - 中上部搜索框。
  - 右上角 Grid/List 切换控件。
  - 下方文章卡片区。
- 文章页：更大的标题区、更精致的 meta 排版、更舒展的正文节奏。

## Visual Language
- 背景色：暖白或浅米色。
- 标题字体：优先使用公开可接入的高质量衬线字体，模拟编辑出版物质感。
- 正文字体/UI 字体：简洁的人文无衬线，强调易读性。
- 卡片：高圆角、浅描边、极轻阴影、上半部分保留插图/色块封面区域。
- 动效：轻微 hover、视图切换和淡入，避免科技产品式炫技。

## Features to Preserve
- 全文搜索
- 标签与分类
- 目录导航
- 代码高亮
- KaTeX
- 暗色模式
- 文章更新时间

## Implementation Strategy
- 重写首页和列表页模板，建立统一的“文章浏览界面”。
- 重写文章详情页结构和样式。
- 使用站点级 partial 和自定义 CSS，不直接修改主题源码。
- 用少量原生 JS 处理 Grid/List 切换和客户端筛选体验。
